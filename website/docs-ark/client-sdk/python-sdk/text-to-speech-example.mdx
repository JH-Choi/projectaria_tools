Using aria.sdk_gen2 to Render Text-to-Speech (TTS) on an Aria Gen2 device.

## Run TTS example
```bash
python ~/Downloads/projectaria_client_sdk_samples_gen2/device_tts.py --text `hello world, my name is Aria Gen 2`
```

## Step-by-step walk through

1. Import Required Modules

Start by importing the necessary modules:
```python
import argparse
import aria.sdk_gen2 as sdk_gen2
```
aria.sdk_gen2 is the SDK for interacting with Aria devices.

2. Parse Command-Line Arguments

Set up argument parsing to allow the user to specify the TTS text:
```python
def parse_args() -> argparse.Namespace:
   parser = argparse.ArgumentParser()
   parser.add_argument(
       "--text",
       dest="text",
       type=str,
       default="",
       required=True,
       help="TTS text to rendered by the device.",
   )
   return parser.parse_args()
```
The --text argument lets the user specify the text to be rendered as speech by the device.
This argument is required.

3. Set Up the Device Client and Configuration

Create and configure the device client:
```python
device_client = sdk_gen2.DeviceClient()
config = sdk_gen2.DeviceClientConfig()
device_client.set_client_config(config)
```
* DeviceClient manages the authentication and device connection.
* DeviceClientConfig is used to specify the target device by serial number or IP address.

4. Connect to the Device

Attempt to connect to the device:
```python
device = device_client.connect()
print(f"Connected to device: {device.connection_id()}")
```
If successful, youâ€™ll see a confirmation message with the device connection ID.

5. Render Text-to-Speech (TTS)

Send the TTS command to the device:
```python
print(f"Rendering TTS: {args.text}")
device.render_tts(text=args.text)
```
The device will render the specified text as speech.
